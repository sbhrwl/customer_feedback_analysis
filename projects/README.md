# Overview
Strategy for NLP models.
* Saving dataset (save_raw_data.py)
* Feature processing
  * Clean data
    * Remove Punctuations
    * Tokenize data
    * Remove Stop words
  * Enhancing dataset
    * Stemming
    * Lemmitisation
  * Create Vectors
    * Count Vectorizer
    * Count Vectorizer with **n** grams
    * TfIdf
  * Add new features
    * Length of Comment
    * Percentage of Punctuations
* Analysis of Features
* Create Word Cloud
* Training Model
* Evaluating Model
  * Generate Confusion Matrix
